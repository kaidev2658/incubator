# Daily Insights

`daily/` stores recurring date-based insight reports generated by the daily-news workflow.

## Purpose

Use this directory for:
- daily news summaries
- recurring insight reports grouped by date
- chronological review of day-by-day external signals

This area is intentionally date-oriented rather than topic-oriented.
If a request becomes a durable topic with follow-up iterations, move that work to `artifacts/` instead of continuing under `daily/`.

## File Layout

Daily reports are stored by month using this convention:

- `daily/YYYY-MM/AgentK_Daily_Insight_YYYY-MM-DD.md`

Examples:
- `daily/2026-03/AgentK_Daily_Insight_2026-03-25.md`
- `daily/2026-02/AgentK_Daily_Insight_2026-02-24.md`

## Operating Rule

- Scheduled daily-news output belongs in `daily/`.
- Date-scoped recurring reporting belongs in `daily/`.
- Topic-specific research, implementation, or evolving project work belongs in `artifacts/`.

## Maintenance Notes

- Add new monthly directories as needed.
- Keep naming consistent so reports remain sortable and indexable.
- If additional recurring report types are introduced later, extend this index with a short subsection rather than creating ad hoc conventions.
