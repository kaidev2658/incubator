# Incubator Docs

Tizen AI-OS, OpenClaw-inspired agent architecture, 그리고 관련 리서치 산출물을 통합 관리하기 위한 문서 포털입니다.

## 빠른 링크
- [Tizen AI-OS PRD](tizen-ai-os-prd.md)
- [Agent Tool Manifest](agents/tool-manifest.md)
- [Research Notes](research/README.md)
- [Roadmaps](roadmaps/README.md)

## 구성
1. **PRD & Strategic Docs** – `tizen-ai-os-prd.md` 와 `agents/` 하위에 기술 요구 사항, tool manifest, security policy 등을 정리합니다.
2. **Research** – 각 논문/분석/제품 조사 결과를 `docs/research/` 하위에 날짜별/주제별로 쌓습니다.
3. **Roadmap + Releases** – `docs/roadmaps/` 에 단계별 목표, 성과지표, 릴리즈 체크리스트를 기록합니다.
4. **Assets** – 다이어그램이나 프레젠테이션에서 쓰일 그림 파일은 `docs/assets/` 에 보관해 GitHub Pages에서 가져다 씁니다.

추가로 필요한 문서 템플릿이나 시각 자료가 있다면 말씀해주세요.
