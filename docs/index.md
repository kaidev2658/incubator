# Incubator Docs

이곳은 여러 기술 주제와 리서치 결과를 수집해두는 문서 포털입니다. 각 주제는 `docs/` 아래 디렉터리로 나뉘며, 필요하다면 템플릿/자산/워크플로를 추가해 확장할 수 있습니다.

## 빠른 링크
- [Tizen AI-OS](tizen-ai-os-prd/README.md)
- [Agent Design](agents/README.md)
- [Research Notes](research/README.md)
- [Roadmaps](roadmaps/README.md)

## 구성
1. **PRD & Strategy** – `/docs/tizen-ai-os-prd`와 향후 추가될 주제 디렉터리를 중심으로 공식 요구사항, 도구 명세, 평가 지표를 정리합니다.
2. **Research Archive** – 조사, 논문 메모, 경쟁 분석 등은 `/docs/research`에 날짜/주제별로 케이스를 남깁니다.
3. **Roadmaps & Releases** – 단계별 목표, KPI, 체크리스트를 `/docs/roadmaps`에 보관합니다.
4. **Assets** – 필요한 다이어그램/이미지/데이터 셋은 `/docs/assets`에 저장하고 문서에서 참조합니다.

필요한 주제가 있으면 말씀만 주세요. 새로운 디렉터리가 필요하면 제가 바로 만들어드릴게요.
