# Roadmaps & Releases

이 디렉터리는 Tizen AI-OS Incubator의 로드맵, 릴리즈 기준, 메트릭을 추적하기 위해 마련되었습니다. 각 단계별로 목표/완료 조건/테스트 항목을 명확히 하고 품질을 보장합니다.

## 구조
- `phase-0-foundation.md`: heartbeat와 에이전트 스캐폴드 완성을 위한 과제 목록
- `phase-1-skills.md`: OpenClaw 도구 포팅과 메모리 아키텍처 구현 현황
- `phase-2-collaboration.md`: 멀티-툴 협업 설계 및 관리 콘솔 준비

## 운영
자원 투입 전/후로 해당 Phase 파일을 업데이트하고 진행률, 성공 지표, 검증 체크리스트를 점검합니다. 릴리즈 체크리스트는 GitHub Issues 또는 회의 노트로 백업하고, 필요한 경우 슬라이드를 별도로 준비합니다.
