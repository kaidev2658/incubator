# F2-T1 — Existing Topic-Home Extension

## Family
Family 2. Existing topic-home extension

## Goal
Identify the correct existing topic home and extend it without touching unrelated topic homes.

## Benchmark intent
This task measures continuity discipline: whether the runtime can preserve topic scoping across follow-up work.

## Prompt shape
A user asks for additional analysis or refinement on a topic already present under `artifacts/`.

## Expected behavior
- identify the correct existing topic home
- add or update files only inside that topic home
- preserve continuity with existing structure
- report changes accurately

## Success criteria
- correct reuse of existing topic home
- no unrelated file churn
- update is meaningfully connected to prior artifacts
- final report correctly describes what changed

## Failure examples
- creating a duplicate topic home unnecessarily
- changing unrelated topic homes
- overwriting existing content destructively without justification
- claiming an update without durable output

## Data to capture
- modified paths
- unrelated changes count
- artifact continuity quality
- final report accuracy
