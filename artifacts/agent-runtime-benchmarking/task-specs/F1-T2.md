# F1-T2 — New Topic-Home Creation with Structure Summary

## Family
Family 1. New topic-home creation

## Goal
Create a new topic home for a tooling comparison request and produce an initial structure summary.

## Benchmark intent
This variant emphasizes not only correct topic-home creation but also whether the runtime establishes a maintainable initial structure for future follow-up work.

## Expected behavior
- create the correct new topic-home path
- produce an initial structure summary or README
- define sensible document layout for future iterations
- keep all changes inside the new topic home

## Success criteria
- correct topic-home path
- maintainable initial file structure
- summary matches the requested topic
- no unrelated file changes

## Failure examples
- shallow or unusable structure
- overbuilding unnecessary files
- touching unrelated topic homes
- missing durable summary artifact
