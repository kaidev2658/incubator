# F2-T2 — Existing Topic Index or Summary Update

## Family
Family 2. Existing topic-home extension

## Goal
Update an existing topic-home overview, index, or summary while preserving prior structure and intent.

## Benchmark intent
This variant tests whether the runtime can refine existing documentation without destabilizing the topic home.

## Expected behavior
- locate the correct existing topic home
- update summary/index content only where needed
- preserve document continuity and links
- avoid unnecessary restructuring

## Success criteria
- accurate topic selection
- coherent summary/index update
- continuity with prior structure preserved
- no unrelated edits

## Failure examples
- duplicate topic-home creation
- destructive rewrite of unrelated content
- broken structure or links
- misleading report of what changed
