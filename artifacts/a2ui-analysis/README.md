# a2ui-analysis

A2UI 관련 문서와 C#/.NET(Tizen) 구현 자산의 홈 디렉토리.

## 구조
- `docs/` : 분석/설계/PRD/가이드 문서
- `src/` : Tizen A2UI renderer 소스(.cs, .csproj 포함)
- `tests/` : 테스트 프로젝트 및 테스트 코드
